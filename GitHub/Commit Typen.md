

| **Typ**  | **Bedeutung**                              | Was hast du gemacht?                       | Beispiel                                       |
| -------- | ------------------------------------------ | ------------------------------------------ | ---------------------------------------------- |
| feat     | Neues feature                              | Neue Funktion / neue Page / neue Component | `feat(auth): add login page`                   |
| fix      | Bugfix                                     | Bug behoben                                | `fix(header): prevent empty form submit`       |
| docs     | Dokumentation                              | Dokumentation                              | `docs(readme): update readme`                  |
| style    | Formatierung, keine funtkionale Änderungen | Nur Formatierung (Prettier, Einrückung)    | `style(ui): format with prettier`              |
| refactor | Code-Umstrukturierung                      | Code umgebaut, ohne Verhalten zu ändern    | `refactor(api): simplify auth service`         |
| test     | Tests hinzugefügt / angepasst              | Tests angelegt oder geändert               | `test(auth): add login tests`                  |
| chore    | Build / CI, interne Aufgaben               | Neue Ordner / Struktur ohne Feature        | `chore(project): add project folder structure` |
| perf     | Performance-Optimierungen                  | Performance-Optimierung                    | `perfimages): optimize image loading`          |
