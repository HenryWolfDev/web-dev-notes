

| **Typ**  | **Bedeutung**                              | Was hast du gemacht?                       | Beispiel                                 |
| -------- | ------------------------------------------ | ------------------------------------------ | ---------------------------------------- |
| feat     | Neues feature                              | Neue Funktion / neue Page / neue Component | `feat(auth): add login page`             |
| fix      | Bugfix                                     | Bug behoben                                | `fix(header): prevent empty form submit` |
| docs     | Dokumentation                              | Dokumentation                              | `docs(readme): update readme`            |
| style    | Formatierung, keine funtkionale Änderungen | Nur Formatierung (Prettier, Einrückung)    | `style(ui): format with prettier`        |
| refactor | Code-Umstrukturierung                      | Code umgebaut, ohne Verhalten zu ändern    |                                          |
| test     | Tests hinzugefügt / angepasst              |                                            |                                          |
| chore    | Build / CI, interne Aufgaben               |                                            |                                          |
| perf     | Performance-Optimierungen                  |                                            |                                          |
